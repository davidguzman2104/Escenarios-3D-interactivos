# Escenarios 3D Interactivos con Three.js

Proyecto desarrollado con **Three.js** para visualizar un escenario 3D interactivo en navegador, con movimiento en primera persona, colisiones, físicas básicas y lanzamiento de esferas.

En esta versión se integró un nuevo escenario en formato **FBX**, sustituyendo el modelo anterior, manteniendo la lógica principal del proyecto.

---

## Descripción

Este proyecto permite al usuario recorrer un entorno 3D interactivo utilizando controles de teclado y mouse. El sistema incluye:

- Cámara en primera persona
- Movimiento con teclado
- Salto
- Colisiones con el escenario
- Lanzamiento de esferas
- Estadísticas de rendimiento
- Escenario 3D cargado con `FBXLoader`

El objetivo principal de la actividad fue **reemplazar el escenario existente por un nuevo modelo descargado desde Sketchfab**, conservando el resto del funcionamiento del proyecto.
